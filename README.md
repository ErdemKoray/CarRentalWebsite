Technologies Used
Frontend: Angular framework is used for creating a dynamic and responsive user interface.
Backend: C# language and .NET Core are used to develop the server-side logic and API endpoints.
Database: PostgreSQL is used to manage and store the rental data, customer information, and booking details.
Main Features
User Registration and Login: Users can create accounts, log in, and manage their profiles.
Car Search and Filter: Users can search for available vehicles based on location, date, and preferences.
Booking Management: Users can book cars, view their booking history, and cancel reservations if needed.
Admin Dashboard: Admins can manage car listings, track rentals, and view customer information.
